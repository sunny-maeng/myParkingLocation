# myParkingLocation
![iOS badge](https://img.shields.io/badge/iOS-14.0%2B-blue)

### 주차위치를 저장하는 서비스를 제공합니다. 
- GPS를 이용해 지도에 주차위치를 기록할 수 있습니다
- GPS수신이 어려운 곳에선 사진이나 스케치를 이용해 위치를 기록할 수 있습니다.
- DarkMode를 지원합니다.
